**Cirrhosis Disease Prediction**


**Overview**

The "Cirrhosis Disease Prediction" project aims to predict the likelihood of a patient having cirrhosis based on various clinical features. The application is built using Django for the backend and frontend, and employs a machine learning model to perform the predictions.

**Features**

User Authentication: Secure login and registration for users.
Patient Data Input: Form to input patient clinical features.
Prediction: Machine learning model to predict the likelihood of cirrhosis.
Results Display: Visualize prediction results and probabilities.
